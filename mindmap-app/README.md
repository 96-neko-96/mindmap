# MindMap Tool - マインドマップツール

個人的なアイデア整理のための高機能Webベースマインドマップツール

![Version](https://img.shields.io/badge/version-1.0.0--phase1-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🎯 プロジェクト概要

このプロジェクトは、React + TypeScript + Tailwind CSSで構築された、モダンで高機能なマインドマップツールです。ブラウザ上で動作し、直感的な操作でアイデアを整理できます。

## ✨ 現在の機能 (Phase 1 - MVP)

### 基本機能
- ✅ **中心ノードの表示** - マインドマップの起点となる中心ノード
- ✅ **子ノードの追加** - ボタンクリックまたはTabキーで子ノードを追加
- ✅ **ノードのテキスト編集** - ダブルクリックでテキストを編集
- ✅ **ノードの削除** - 選択してDeleteキーまたはサイドパネルから削除
- ✅ **ノードの選択** - クリックで選択、選択状態を視覚的に表示

### ビジュアル機能
- ✅ **SVG接続線** - 親ノードと子ノード間を美しい曲線で接続
- ✅ **ノードのカスタマイズ** - 色、形状、サイズなどをカスタマイズ可能
- ✅ **レスポンシブデザイン** - モダンでクリーンなUIデザイン

### 操作機能
- ✅ **ドラッグ&ドロップ** - ノードを自由に移動
- ✅ **キャンバスのパン** - キャンバスをドラッグで移動
- ✅ **ズーム機能** - マウスホイールまたはツールバーボタンでズーム
- ✅ **キーボードショートカット**
  - `Tab` - 子ノードを追加
  - `Delete` / `Backspace` - ノードを削除
  - `Enter` - テキスト編集を確定
  - `Escape` - テキスト編集をキャンセル

## 🛠️ 技術スタック

- **フレームワーク**: React 18 with Hooks
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **ビルドツール**: Vite
- **状態管理**: React Context API + useReducer

## 📦 インストール

### 前提条件
- Node.js 18.0以上
- npm または yarn

### セットアップ手順

1. **リポジトリをクローン**
   ```bash
   git clone <repository-url>
   cd mindmap/mindmap-app
   ```

2. **依存関係をインストール**
   ```bash
   npm install
   ```

3. **開発サーバーを起動**
   ```bash
   npm run dev
   ```

4. **ブラウザでアクセス**
   ```
   http://localhost:5173
   ```

## 🎮 使い方

### 基本操作

1. **ノードの追加**
   - ノードを選択してサイドパネルの「Add Child Node」ボタンをクリック
   - または選択した状態で`Tab`キーを押す

2. **ノードの編集**
   - ノードをダブルクリックしてテキスト編集モードに入る
   - テキストを入力して`Enter`で確定、`Escape`でキャンセル

3. **ノードの移動**
   - ノードをドラッグ&ドロップで自由に配置

4. **ノードの削除**
   - ノードを選択して`Delete`キーを押す
   - またはサイドパネルの「Delete Node」ボタンをクリック

5. **ビューの操作**
   - マウスホイールでズームイン/アウト
   - キャンバスの空白部分をドラッグしてパン（移動）
   - ツールバーの「Reset View」でビューをリセット

### ショートカットキー一覧

| キー | 動作 |
|------|------|
| `Tab` | 子ノードを追加 |
| `Delete` / `Backspace` | 選択中のノードを削除 |
| `Enter` | テキスト編集を確定 |
| `Escape` | テキスト編集をキャンセル |
| マウスホイール | ズームイン/アウト |
